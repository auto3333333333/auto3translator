# 🌐 おーととらんすれーたー (Auto 3 Translator)

[![Version](https://img.shields.io/badge/version-v1.1.0-blue.svg)](https://github.com/auto3333333333/auto3translator/releases/latest)
[![Platform](https://img.shields.io/badge/platform-Windows%2010%20%2F%2011-lightgrey.svg)]()

選択したテキストをショートカットキー一発で自動コピーし、Google翻訳経由で翻訳して元のテキストと瞬時に置き換える、Windows向けの超軽量・高速バックグラウンド自動翻訳ツールです。

Discordでの海外コミュニティとのチャット、海外ゲームのチャット欄、各種ドキュメントの作成などを劇的にスムーズにします。

---

## ✨ 主な機能

* **一発で翻訳・置換**: 文字を選択してショートカットを押すだけで、`コピー ➔ 翻訳 ➔ 貼り付け` が完全自動で完結します。
* **双方向翻訳に対応**: 
  * `日本語 ➔ 外国語`（デフォルト: `Alt + F5`）
  * `外国語 ➔ 日本語`（デフォルト: `Alt + F6`）
* **堅牢なフックシステム**: 特殊な配列のキーボードや、特定のゲーム内でも入力を逃さない最下層キー監視（pynputベース）を採用。
* **すり抜け防止ウェイト**: 人間らしい入力速度（タメ）を再現し、Discord等のアプリ側でのコピペ認識エラーを徹底排除。
* **タスクトレイ常駐型**: 起動後はタスクトレイに格納され、作業の邪魔になりません。
* **自動アップデート機能**: 起動時に自動で最新リリースをチェック。手動での確認ボタンも搭載。
* **アイコン自動修復**: フォルダ内に `icon.ico` がなくても、起動時にGitHubから自動補填します。

## 🚀 使い方

### 1. 起動
ダウンロードした `おーととらんすれーたー.exe` をダブルクリックして起動します。
> 💡 **ポイント**: 常に安定して動作させるため、自動的に「管理者権限」を要求（UAC画面が表示）されます。「はい」を選択して起動してください。

### 2. 基本操作
* **日本語を外国語に翻訳する**:
  1. 翻訳したい日本語の文章をマウス等で選択（反転）させます。
  2. `Alt + F5` を押します。
  3. 選択していた文章が、一瞬で設定された外国語に書き換わります。
* **外国語を日本語に翻訳する**:
  1. 読み方のわからない外国語の文章を選択します。
  2. `Alt + F6` を押します。
  3. 選択していた文章が自動で日本語に書き換わります。

### 3. 設定の変更
タスクトレイにあるアプリのアイコンを右クリックし、**「⚙️ 設定を開く」** を選択するか、バックグラウンド移行時の画面から以下の項目をカスタマイズできます。
* 翻訳先の言語設定（英語、韓国語、中国語、フランス語、スペイン語など）
* 各種ショートカットキーの変更
* 翻訳完了時のデスクトップ通知（ポップアップ）のON/OFF

## 📋 動作要件

* **OS**: Windows 10 / 11 (64bit環境)
* **権限**: 管理者権限（UACの許可が必要です）
* **ネットワーク**: 翻訳処理、およびアップデート・アイコン取得のためにインターネット接続が必要です。

## ⚠️ トラブルシューティング

### Q. 「WindowsによってPCが保護されました」と表示されて起動できない
自作のexeファイルであるため、WindowsのSmartScreen機能が警告を出す場合があります。
画面内の **「詳細情報」** をクリックし、右下に出現する **「実行」** ボタンを押すことで次回から正常に起動できるようになります。

### Q. 特定のゲームやアプリの上でショートカットが反応しない
操作対象のアプリが「管理者として実行」されている場合、本ツールも管理者権限で起動している必要があります。本ツールはデフォルトで管理者権限を要求する仕様になっていますが、上手くいかない場合はexeファイルを右クリックして「管理者として実行」を明示的に選択してください。
